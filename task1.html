1. Tạo một trang HTML với tiêu đề là "Responsive Footer Exercise". 2. Sử dụng CSS để áp dụng các kiểu cho trang web: - Thiết lập font chữ mặc định là Arial hoặc font chữ không-serif khác. - Đặt màu nền của thẻ footer là #333 (màu đen). - Đặt màu chữ của thẻ footer là #fff (màu trắng). - Đặt padding cho thẻ footer là 20px. 3. Áp dụng responsive layout sử dụng media queries: - Đối với màn hình có chiều rộng lớn hơn hoặc bằng 992px: + Sử dụng CSS Grid để hiển thị 4 cột ngang bằng nhau. + Thiết lập thuộc tính display là grid cho phần tử có class "footer-columns". + Sử dụng grid-template-columns để tạo 4 cột bằng nhau, mỗi cột chiếm 1fr (phần bằng nhau). + Đặt khoảng cách giữa các cột bằng 20px sử dụng grid-gap. - Đối với màn hình có chiều rộng từ 576px đến 991px: + Sử dụng CSS Grid để hiển thị 2 cột ngang bằng nhau. + Thiết lập thuộc tính display là grid cho phần tử có class "footer-columns". + Sử dụng grid-template-columns để tạo 2 cột bằng nhau, mỗi cột chiếm 1fr (phần bằng nhau). + Đặt khoảng cách giữa các cột bằng 20px sử dụng grid-gap. 4. Trên trang HTML, trong thẻ footer, tạo một div có class "footer-columns". 5.
